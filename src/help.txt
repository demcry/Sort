HELP
Для того, чтобы отсортировать текстовый файл, необходимо ввести
	sorting [option/combo option] [textfile]
Список параметров:
	-a/--alphabet - Сортировка строк в алфавитном порядке
	-r/--reverse - Сортировка строк по алфавиту в обратном порядке
	-i/--ignore_case - Сортировка строк без учета регистра
	-R/--random - Сортировка строк в случайном порядке
	-v/--version - Запрос версии
	-h/--help - Запрос помощи
	-a -- ignore_case/-i --alphabet/--alphabet --ignore_case/--ignore_case --alphabet/-a-i/-i-a/-ai/-ia - Сортировка по алфавиту без учета регистра
	-r --ignore_case/-i --reverse/-r-i/-i-r/-ri/-ir -Сортировка по алфавиту в обратном порядке без учета регистра
Если не введены параметры - автоматически устанавливается сортировка по алфавиту и программа ожидает ввода пользователем строк. Для завершения ввода нажмите три раза "L.Ctrl+L.Shift+D"
